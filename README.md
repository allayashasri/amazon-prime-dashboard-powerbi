# 📊 Amazon Prime Dashboard (Power BI)

An interactive data analysis dashboard built using Power BI to analyze Amazon Prime content and extract meaningful insights.

---

## 🚀 Features
- Analyzed 9,000+ records of Amazon Prime titles
- Created KPIs: Total Titles, Movies vs TV Shows, Top Genres
- Visualized trends by release year, country, and ratings
- Interactive filters for dynamic exploration

---

## 🛠️ Tools Used
- Power BI
- Excel
- Data Cleaning (Power Query)

---

## 📌 Key Insights
- Movies dominate over TV Shows
- Content growth increased after 2015
- USA and India are top content producers
